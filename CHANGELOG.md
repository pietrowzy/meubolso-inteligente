## [0.1.0] - 19/05/2026 - Em desenvolvimento

### Planejado
- Estrutura inicial do projeto MeuBolso Inteligente.
- Arquitetura backend usando Flask.
- Banco de dados MySQL.
- Dashboard financeiro.
- Sistema de receitas e despesas.
- Relatórios financeiros.
- Integração com IA usando Groq.
- Sistema de feedback dos usuários.

### Definido
- Organização inicial de pastas.
- Uso de HTML, CSS e JavaScript puro no frontend.
- Uso de Flask no backend.
- Uso de sessões para autenticação.
- Uso de Chart.js para gráficos.

### Não implementado nesta fase
- Módulo de metas financeiras.
